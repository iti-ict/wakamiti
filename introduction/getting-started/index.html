      Primeros pasos

      1. Instala Wakamiti

      Descarga la imagen docker de Wakamiti:

      docker pull kukumo/kukumo

      2. Ejecuta los tests

      Ubícate en el directorio que contiene los tests de Wakamiti y lánzalo:

      cd ~/test
docker run --rm -it -v "$(pwd):/kukumo" kukumo/kukumo
